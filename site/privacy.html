Privacy Policy

Cue is designed to keep user data on-device whenever possible.

Data used by the app

Cue uses Real-Debrid's official OAuth device flow and stores authentication state locally. Android uses encrypted storage. iOS uses Keychain-backed storage.

Cue reads Real-Debrid account status from the authenticated /rest/1.0/user endpoint to show premium state, expiry date, and reminder timing.

Reminder preferences and diagnostics previews stay on-device. Diagnostics export is manual and is not uploaded anywhere by Cue.

Disconnect removes local Cue authentication state and reminder artifacts. Cue does not create a separate backend account.

Data not collected

Cue does not include analytics, tracking, ad SDKs, or third-party crash reporting in v1.

Network behavior

Cue talks directly to the official Real-Debrid API hosts. There is no Cue backend.

Independence

Cue is an independent app for Real-Debrid users and is not affiliated with, endorsed by, or operated by Real-Debrid.
